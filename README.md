- Node.js 18+

### Installation

1. **Clone the repository**
   \`\`\`bash
   git clone https://github.com/BandaAkshith/my-portfolio.git
   cd my-portfolio
   \`\`\`

2. **Install dependencies**
   \`\`\`bash

   \`\`\`

   \`\`\`bash
